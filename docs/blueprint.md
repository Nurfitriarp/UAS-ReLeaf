# **App Name**: ReLeaf

## Core Features:

- User Profiles: Allow both psychiatrists and patients to create and manage their profiles, including personal information, qualifications, and areas of expertise.
- Psychiatrist Discovery: Enable patients to find suitable psychiatrists based on criteria such as specialization, location, and availability.
- Appointment Booking: Allow patients to book consultation appointments with psychiatrists through the platform.
- Mental Health Articles: Provide a curated collection of informative articles on various mental health topics.
- Community Forum: Implement a community forum where users can find and join communities based on shared interests and experiences.
- AI-Powered Psychiatrist Matching: Use a generative AI tool to analyze patient profiles and preferences, then suggest potential psychiatrist matches.
- Sign In with Google: Implement the latest 'Sign In with Google' (Google Identity Services)/Google OAuth for secure and easy user authentication.

## Style Guidelines:

- Primary color: A muted, natural green (#8FBC8F) to convey calmness and health.
- Background color: A very light off-white (#F5F5DC) to provide a soft, unobtrusive backdrop.
- Accent color: A gentle, desaturated blue (#ADD8E6) to create a sense of trust and tranquility.
- Body and headline font: 'PT Sans' for a modern yet warm and accessible feel.
- Use simple, nature-inspired icons with rounded edges to complement the ReLeaf brand. Incorporate the green leaf logo subtly throughout the interface.
- Maintain a clean and intuitive layout with ample whitespace to reduce visual clutter and enhance user experience.
- Use subtle animations and transitions to provide feedback and guide users through the application.
